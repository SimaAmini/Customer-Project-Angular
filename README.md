# customer-project


## Installation


### Database
create a database named `testdb` on mysql

### Run back-end
- `cd back-end`
- `npm install` to install the application's dependencies
- `npm start` to start the application

### Run front-end
- `cd front-end`
- `npm install` to install the application's dependencies
- `ng serve` to start the application
-  Navigate to `http://localhost:4200/`
